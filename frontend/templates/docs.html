How Vanguard Works

1
Crawling

Discovers all pages, forms, and endpoints of the target website

2

Computes real CVSS scores with temporal and environmental metrics

4

Scan Modes

Mode Intensity What It Scans Use Case
Ultra-Safe Minimal
  • Basic misconfig checks
  • Sensitive file detection
  • Headers analysis (selective)
 Large public sites (Google, Facebook, etc.) - Very minimal impact
Safe Low
  • All above
  • Quick SQL injection tests
  • Quick XSS tests
 Fast scans with reduced checks - Minimal false positives
Standard Medium
  • All above
  • Full security headers analysis
  • Comprehensive SQL injection detection
  • Context-aware XSS testing
  • Directory listing checks
  • CORS misconfiguration
  • Cookie security flags
  • CSRF token checks
 Recommended - Balanced coverage
Aggressive High
  • Everything in Standard
  • Open redirect testing
  • URL parameter fuzzing
  • Extended payload testing
  • Context breaking tests
  • All XSS variants
  • All SQL injection types
 Thorough testing - May have some false positives
